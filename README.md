Steam Games SQL Analysis
A data analysis project using PostgreSQL and to explore and analyze a dataset of 1,000 Steam games released up to 2026.
Dataset

Source: Steam Games Dataset
Size: 1,000 games
Columns: AppID, Name, Release Date, Primary Genre, All Tags, Price, Discount, Review Score, Total Reviews, Estimated Owners, Peak Players

Skills Demonstrated

Data exploration and aggregation
CASE WHEN categorization
Window Functions and CTEs
JOINs across multiple tables
Subqueries
Data Cleaning and NULL handling
Views
String Parsing with UNNEST

Key Questions Answered

Which genre has the most players and highest average price?
Which games have a loyal fanbase based on review score?
Which games are overperforming in their genre?
What are the most common tags across all games?
Which genres generate the highest estimated revenue?

How to Run

Create the tables using the CREATE TABLE statements at the top of the file
Import the dataset into steam_games
Run any query from the .sql file in PostgreSQL
